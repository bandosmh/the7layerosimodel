Data Link Layer (Layer 2)

Definition

The Data Link Layer handles node-to-node data delivery on a local network segment, managing physical addressing, error detection, and flow control.

Frame Structure

Preamble

Synchronization

MAC Header

Addresses

Data

Core Functions

  • Framing: Packaging data into frames
  • Physical Addressing: MAC address management
  • Error Control: Detection and correction
  • Flow Control: Managing data transmission rate
  • Access Control: Managing media access

Protocols and Standards

  • Ethernet: Most common LAN technology
  • Wi-Fi (802.11): Wireless networking
  • PPP: Point-to-Point Protocol
  • HDLC: High-level Data Link Control

Network Devices

  • Switches: Intelligent frame forwarding
  • Bridges: Connect network segments
  • Network Interface Cards: Physical network access
  • Wireless Access Points: Wireless connectivity
