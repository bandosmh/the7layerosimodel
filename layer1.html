Physical Layer (Layer 1)

Definition

The Physical Layer defines the electrical, mechanical, and functional characteristics of the physical network, handling the transmission of raw bits over a physical medium.

Core Functions

  • Bit Transmission: Converting data to signals
  • Physical Topology: Network layout design
  • Transmission Mode: Simplex, half-duplex, full-duplex
  • Line Configuration: Point-to-point and multipoint

Transmission Media

Copper Cables

  • UTP/STP
  • Coaxial
  • Electrical signals

Fiber Optics

  • Single-mode
  • Multi-mode
  • Light signals

Wireless

  • Radio waves
  • Microwaves
  • Infrared

Hardware Components

  • Network Interface Cards (NICs): Network connectivity
  • Repeaters: Signal amplification
  • Hubs: Physical layer connectivity
  • Cables and Connectors: Physical medium

Signal Characteristics

  • Bandwidth: Data transmission capacity
  • Frequency: Signal oscillation rate
  • Attenuation: Signal loss over distance
  • Noise: Interference management
