Network Layer (Layer 3)

Definition

The Network Layer handles logical addressing and routing of data packets between different networks, determining the optimal path for data transmission.

Core Functions

  • Logical Addressing: IP address assignment and management
  • Routing: Determining the best path for data
  • Packet Forwarding: Moving packets between networks
  • Fragmentation: Breaking packets into smaller units

IP Protocol Versions

IPv4

  • 32-bit addresses
  • 4 billion addresses
  • Widely deployed
  • Uses NAT for address shortage

IPv6

  • 128-bit addresses
  • Vast address space
  • Built-in security
  • Better performance

Routing Protocols

  • OSPF: Open Shortest Path First
  • BGP: Border Gateway Protocol
  • RIP: Routing Information Protocol
  • EIGRP: Enhanced Interior Gateway Routing Protocol

Network Devices

  • Routers: Direct traffic between networks
  • Layer 3 Switches: Combined switching and routing
  • Firewalls: Network security devices
  • Multilayer Switches: Enterprise network devices
